## 💪🏻 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) >= 16
- [npm](https://npmjs.com/) >= 8
- Supported OSes: Linux, macOS and Windows

### Installation

```sh
# Install dependencies
npm install
```

### Usage

```sh
cp .env.example .env
```

edit the `API_BASE_URL` to the real address

```sh
# development model
npm run dev
```
